# **App Name**: UPI Linker

## Core Features:

- UPI Link Generation: Generates a UPI payment link based on the entered UPI ID and amount.
- QR Code Generation: Generate QR code equivalent of UPI payment link for easy sharing.
- Payment Request Storage: Saves the payment request details, including the UPI ID, amount, and timestamp, in Firestore.
- Payment Status Tracking: Monitors and updates the payment status using Firestore real-time listeners.
- Payment History: View logged in user's UPI payment requests that have been created.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for reliability and trust in financial transactions.
- Background color: Light cyan (#E0FFFF), a very pale tint of the primary to establish the cool, airy feeling.
- Accent color: Lime green (#32CD32) to indicate successful payments or completed transactions, symbolizing a positive financial outcome.
- Font: 'Inter', a sans-serif, for clear readability of UPI IDs and amounts.
- Use minimalist icons for payment options, history, and settings.
- Simple, single-column layout for easy navigation.
